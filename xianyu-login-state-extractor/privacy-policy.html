闲鱼登录态提取器隐私说明

版本：2026-04-01

用途

本扩展唯一用途是在用户主动点击“提取登录态 JSON”时，读取当前闲鱼页面的登录态 Cookie、请求头和浏览器环境信息，并生成本地 JSON 供用户自行复制和保存。

收集内容

  • 当前闲鱼页面 Cookie，包括 `HttpOnly` Cookie。
  • 当前页面 URL、来源页、页面可见状态。
  • 浏览器环境信息，例如 User-Agent、语言、时区、屏幕尺寸、硬件并发数。
  • 当前域名下的 `localStorage` 和 `sessionStorage` 快照。
  • 一次真实请求中可观察到的常用请求头。

数据使用方式

  • 所有数据仅在本地浏览器中处理。
  • 扩展不会自动上传数据到任何服务器。
  • 复制到剪贴板仅在用户主动点击时发生。

权限说明

  • cookies：读取闲鱼站点 Cookie。
  • scripting：读取页面环境和 Web Storage。
  • webRequest：观察一次页面请求头。
  • activeTab / tabs：定位当前活动标签页。

用户控制

  • 只有用户点击提取按钮时才会触发采集。
  • 用户可以随时停用或卸载扩展。

风险提示

登录态 JSON 属于敏感身份信息，请只在可信设备和可信系统中使用，不要随意分享给他人。
